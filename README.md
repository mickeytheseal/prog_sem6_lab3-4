# prog_sem6_lab3-4
 Программирование на языке высокого уровня. Семестр 6. Лабораторные работы №3-4
## Задание
 Необходимо реализовать иерархию классов:<br/>
 ![Иерархия](https://github.com/mickeytheseal/prog_sem6_lab3-4/blob/main/lab3.PNG)<br/>
 Для каждого класса определить свои особенности, например для класса
Абитуриент особенностью будет наличие вступительных баллов или
возможность выбрать несколько профилей поступления
Для студенты особенностью будет допустим наличие зачетной книжки и
студенческого билета.
Для бюджетных и внебюджетных студентов соответственно особенностью
будет наличие договора, необходимости оплачивать обучение или наоборот
получение стипендии.<br/>
 Используя механизмы преобразования типов, необходимо реализовать
возможность отработки следующего сценария:
1. В систему(программу) заводится абитуриент
2. Далее после поступления «абитуриент» становится студентом
3. Так как «абитуриенту» не хватило баллов для поступления на бюджет с
ним был заключен договор об оказании платных услуг, и он стал
внебюджетным студентом
4. Через год обучения студент был переведен на бюджетное место.
