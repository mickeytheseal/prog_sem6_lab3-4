# prog_sem6_lab3-4
 Программирование на языке высокого уровня. Семестр 6. Лабораторные работы №3-4
## Задание на лабораторную работу №3
 Необходимо реализовать иерархию классов:<br/>
 ![Иерархия](https://github.com/mickeytheseal/prog_sem6_lab3-4/blob/main/lab3.PNG)<br/>
 Для каждого класса определить свои особенности, например для класса
абитуриента особенностью будет наличие вступительных баллов или
возможность выбрать несколько профилей поступления.
Для студента особенностью будет допустим наличие зачетной книжки и
студенческого билета.
Для бюджетных и внебюджетных студентов соответственно особенностью
будет наличие договора, необходимости оплачивать обучение или наоборот
получение стипендии.<br/>
 Используя механизмы преобразования типов, необходимо реализовать
возможность отработки следующего сценария:
1. В систему(программу) заводится абитуриент
2. Далее после поступления «абитуриент» становится студентом
3. Так как «абитуриенту» не хватило баллов для поступления на бюджет с
ним был заключен договор об оказании платных услуг, и он стал
внебюджетным студентом
4. Через год обучения студент был переведен на бюджетное место.
## Задание на лабораторную работу №4
### Часть 5
Доработайте прошлое задание реализовав возможность хранения
задолженностей и дисциплин текущего семестра, в специальной структуре на
свой выбор. Добавьте метод позволяющий зафиксировать факт сдачи
задолженности, путем удаления ее из списка задолженностей с фиксацией
оценки. Так же необходимо реализовать метод фиксации сдачи текущих
дисциплин, при этом можно подумать над тем, чтобы объединить эти два
метода.<br/>
### Часть 6
Необходимо доработать программу из прошлого задания, определив разницу
в поведении объектов(студентов) в ситуации перевода на старший курс.
Следует напомнить, что общая процедура перевода на старший курс
заключается в том, что студент может быть переведен только в случае наличия
не более 3 долгов за последний семестр, но есть разница при переводе
внебюджетных студентов так как у них так же проверяется наличие
задолженностей по оплате.<br/>
Для выполнения данного задания необходимо определить метод перевода на
старший курс, при этом для класса вне бюджетников необходимо метод
переопределить с учетом специфики их обучения.
